# Privacy Policy — FlightCheck Pro

**Last updated: May 2026**

## Overview

FlightCheck Pro is developed by Tom Buckley. This privacy policy explains how the app handles your data.

## Data We Collect

FlightCheck Pro stores the following data **locally on your device only**:

- Flight log entries (dates, times, aircraft, Hobbs readings)
- Defect log entries
- Currency tracker records
- Weight & balance calculations
- Checklist completion state
- App preferences and settings

**None of this data is transmitted to any server or third party.**

## Checklist Scanner

The checklist scanner feature sends images you select to the Anthropic Claude API for processing. Images are used solely to extract checklist data and are not stored or used for any other purpose. No personally identifiable information is included in these requests.

## Data Storage

All app data is stored locally on your device using standard iOS storage. Data is not backed up to any external server by this app. You can delete all app data at any time via Settings → Reset App Data.

## Third Party Services

- **Anthropic Claude API** — used by the checklist scanner feature only
- **Aviation weather data** — fetched from aviationweather.gov (public API, no personal data sent)
- **Wind data** — fetched from open-meteo.com (public API, no personal data sent)

## Children's Privacy

This app is not directed at children under 13.
